# LocalSend 介绍
![logo](https://localsend.org/img/screenshot-iphone.webp)

LocalSend 是一个免费、开源、跨平台的局域网文件互传工具，无需互联网连接，即开即用，支持 Windows、macOS、Linux，以及 iOS、Android。

LocalSend 通过 P2P 技术实现设备间直接传输，保证了用户数据的隐私性和安全性。
## 官网
[官网](https://localsend.org/#/) :eyes:

## 功能
- 在局域网内任意设备之间传输文件和消息
- 支持文件、图片、视频、音频等多种文件格式
- 传输速度快，不受网络影响
- 安全可靠，无需担心数据泄露

## 使用方法
下载并安装 LocalSend

在所有需要互传文件的设备上启动 LocalSend

在发送方设备上选择需要发送的文件或消息，然后点击“发送”

在接收方设备上选择接收文件或消息，然后点击“接收”

## 使用示例
### 发送文件
在发送方设备上，打开 LocalSend 应用程序。

在“文件”选项卡中，选择需要发送的文件。

在“操作”栏中，点击“发送”。

在接收方设备上，打开 LocalSend 应用程序。

在“接收”选项卡中，选择接收文件。

### 发送消息
在发送方设备上，打开 LocalSend 应用程序。

在“消息”选项卡中，输入需要发送的消息。

在“操作”栏中，点击“发送”。

在接收方设备上，打开 LocalSend 应用程序。

在“接收”选项卡中，查看消息。
## 常见问题
### 如何查看设备名称？
在 LocalSend 应用程序中，点击“设置”按钮，然后在“设备名称”栏中输入设备名称。

### 如何设置密码？
在 LocalSend 应用程序中，点击“设置”按钮，然后在“密码”栏中输入密码。

### 如何禁用自动发现？
在 LocalSend 应用程序中，点击“设置”按钮，然后在“自动发现”栏中取消勾选“启用自动发现”。
